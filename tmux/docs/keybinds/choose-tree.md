## Choose Tree TMUX Keybindings

- Enter - Select item
- "x" - Kill item
- "X" - Kill tagged items
- "<" - Scroll list left
- ">" Scroll list right
- "C-s" - Search by name
- "n" - Repeat last search
- "t" - Toggle tag
- "T" - Tag none
- "C-t" - Tag all
- "q" or Escape - Cancel/exit
- "?" - Show keys
