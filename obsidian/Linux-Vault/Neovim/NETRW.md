### Basic Navigation

- **Enter**: Open the selected directory or file with the default action.
    
- **- (minus)**: Go up to the parent directory.
    

### File Operations

- **%**: Create a new file in the current directory. You will be prompted to enter a file name.
    
- **d**: Create a new directory in the current directory. You will be prompted to enter a directory name.
    

### Opening Files

- **o**: Open the file in a horizontal split window.
    
- **v**: Open the file in a vertical split window.
    
- **t**: Open the file in a new tab.
    
- **P**: Go to the parent directory.
    

### File and Directory Manipulation

- **R**: Rename the selected file or directory.
    
- **D**: Delete the selected file or directory (be careful with this; confirmations are enabled by default).
    
- **X**: Execute a shell command in the selected directory.
    

### Browsing and Sorting Options

- **i**: Cycle through listing formats (thin, long, wide, tree).
    
- **s**: Select and cycle through sorting options (name, time, size).
    
- **r**: Reverse the sorting order.
    
- **gh**: Toggle showing/hiding hidden files.
    

### Other Useful Commands

- **Ctrl-^ (Ctrl-6)**: Go to the previous directory (like a back button).
    
- **x**: View the file using the system's default application.
    
- **q**: Close the netrw window or tab.
    

### Bookmarking

- **mb**: Bookmark the current directory.
    
- **gb**: Open a list of bookmarks. You can navigate to any bookmarked directory by selecting it.
