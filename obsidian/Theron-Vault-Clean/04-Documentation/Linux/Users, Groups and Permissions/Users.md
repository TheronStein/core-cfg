## Commands




* Rename User 
```Bash
usermod -l new_username old_username
```

* Rename User Home Directory  
```Bash
usermod -d /home/new_username -m new_username
```

* Show available Users
```Bash
cut -d: -f1 /etc/passwd

Example Output:

Rampage
Zandronum
Steam
Chaoscore
Renegades
Postgres
zandronum
steam
```
