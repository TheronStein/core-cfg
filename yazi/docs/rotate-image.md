# rotate-image.yazi

[](https://github.com/tasnimAlam/rotate-image.yazi#rotate-imageyazi)

## Requirements

[](https://github.com/tasnimAlam/rotate-image.yazi#requirements)

[imagemagick](https://github.com/ImageMagick/ImageMagick)

## Installation

[](https://github.com/tasnimAlam/rotate-image.yazi#installation)

ya pack -a tasnimAlam/rotate-image

## Usage

[](https://github.com/tasnimAlam/rotate-image.yazi#usage)

Add this to your `~/.config/yazi/keymap.toml`:

\[\[manager.prepend_keymap\]\]
on   = \["R"\]
run  = "plugin rotate-image"
desc = "Rotate images"
