## Requirements

[](https://github.com/tasnimAlam/thumbnail.yazi#requirements)

[swayimg](https://github.com/artemsen/swayimg)

## Installation

[](https://github.com/tasnimAlam/thumbnail.yazi#installation)

ya pack -a tasnimAlam/thumbnail

## Usage

[](https://github.com/tasnimAlam/thumbnail.yazi#usage)

Add this to your `~/.config/yazi/keymap.toml`:

\[\[manager.prepend_keymap\]\]
on   = \[ "&lt;C-t&gt;" \]
run  = 'plugin thumbnail'
desc = "Open current directory in Swayimg gallery"
